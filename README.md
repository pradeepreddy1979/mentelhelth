# 🧠 Mental Health Condition Classification using NLP

## 📌 Project Overview

This project builds a Natural Language Processing (NLP) based machine
learning system to classify mental health conditions from social media
text. The model analyzes user-generated text and predicts the mental
health category.

The goal of the project is to assist in early detection of mental health
signals using AI and text analytics.

------------------------------------------------------------------------

## 🎯 Objectives

-   Detect mental health conditions from textual data
-   Build an NLP based machine learning classifier
-   Analyze social media text for mental health signals
-   Evaluate model performance using classification metrics

------------------------------------------------------------------------

## 📂 Dataset

The dataset contains social media posts labeled with mental health
conditions.

### Columns

  Column   Description
  -------- ----------------------------------
  text     Social media text / tweet
  label    Mental health condition category

Example categories: - Depression - Anxiety - Suicidal - Normal
